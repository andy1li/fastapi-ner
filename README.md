# fastapi-ner

Follow along [TDD Full Course](https://youtu.be/eAPmXQ0dC7Q), but using `fastapi` and `pytest` instead of `flask` and `unittest`.



Run server:

```shell
uvicorn main:app --reload
```

Go to browser:

```shell
http://localhost:8000
```

Run tests in top directory:

```shell
python -m pytest -v
```
